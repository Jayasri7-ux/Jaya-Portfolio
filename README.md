# Jaya Sri Palli - Portfolio Website

A modern, clean, and minimal portfolio website built with React.

## Features
- **Responsive Design**: Works on Desktop, Tablet, and Mobile.
- **Modern UI**: Clean typography, purple accent, and smooth hover effects.
- **Sections**: Hero, About (Education Timeline), Skills, Experience, Projects, Services, and Contact.

## How to Run the Project

### Prerequisites
- [Node.js](https://nodejs.org/) (which includes `npm`)

### Steps to Run Locally

1. **Install Dependencies**:
   Open a terminal in the project root and run:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm start
   ```
   The website will be available at `http://localhost:3000`.

### Build for Production
To create an optimized production build in the `build` folder:
```bash
npm run build
```

## Project Structure
- `src/components/`: Individual UI sections.
- `src/styles/`: Global variables and component styles.
- `public/`: Static assets and the main HTML entry.
